# tenkey-meetup/raffle-v2

[天キー](https://tenkey.tokyo/ja/)用抽選システムv2

## ビルド・起動方法
1. PodmanとPodman-Composeをインストールします。
1. Podmanコンテナをビルドします。（`./build-docker-images.sh`）
1. フロントエンド・バックエンドコンテナともに起動します。(`podman compose up`)
1. `http://<ip>:9001` にアクセスします。

## 利用方法
以下の2つのCSVファイルを編集メニューからアップロードしてください。
- Connpassフォーマットの参加者リスト（イベントページ→申込者を管理する→CSVダウンロード）
  - 手動で作製する場合、`ユーザー名`、`表示名`、`参加ステータス`（「参加」または「参加キャンセル」）、`受付番号`（ユニークキー）の列が必要です
- 景品リスト
  - `管理No`（ユニークキー）、`景品名`、`提供元`の列が必要です
  
バーコード読み取りには、サフィックスをエンターキーに設定したバーコードリーダーをご利用ください。  
ハードウェアボタンで抽選を行う場合、キーマップを`n`（次へ・確定）と`r`（再抽選）キーに設定してください。
